# Консольный сапер на С++
### Создание консольного приложения на С++ в объектно-ориентированном стиле

### Разработчик:
- Степанюк Ксения Максимовна

### Цель:
- Реализовать классическую игру "Сапер" на языке программирования С++

### Задачи:
- Реализовать классы и методы в них
- Вызвать методы и проверить их работу
- Создание маски чтобы закрыть неоткрытые ячейки
- Реализовать стек дл яоткрытие смежных пустых ячеек до границ и цифр
- Окрасить состовляющие игры для читаемости
